# dex-sdk
A dex(Decentralized Exchange) that supports the uniswap v2 contract variant
- [x] BSC(Pancake Swap V2)
- [x] ETH(Uniswap V2)

# Installation 
Latest version
```shell
go get github.com/Jonescy/dex-sdk
```
Or Special version
```shell
go get github.com/Jonescy/dex-sdk@v0.1.0
```
